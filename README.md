## 📅 추진 일정

### Phase 1: 준비 (1-2개월)
- 정부지원금 신청 (농림축산식품부)
- 토지 확보
- 기술 파트너 선정

### Phase 2: 시공 (2-3개월)
- 온실 시설 구축
- 스마트팜 장비 설치
- 수경재배 시스템 구성

### Phase 3: 시운전 (1-2개월)
- 운영 인력 교육
- 베타 운영 및 테스트
- 초기 고객 확보

### Phase 4: 본격 운영 (6개월+)
- 정상 생산 및 판매
- 매출 창출
- 확장 계획 추진
